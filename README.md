## Features

### 🎨 Beautify
Wrap any screenshot in a gradient background, app frame, rounded corners, and drop shadow. Looks good in one click.

### ✏️ Annotate
Arrows, callouts, numbered steps, highlights, text, speech bubbles, ruler, emoji — 16 tools with colour and stroke controls.

### 🔒 Redact
Blur, pixelate, or solid-redact sensitive areas. Solid redact is the only truly safe option for passwords and IDs.

### 🔤 OCR — Extract Text
Drag to select any region and pull out the text. Works fully offline using an on-device model. No API, no account, nothing leaves your machine.

### 🖼 App Frames
Wrap your screenshot in a macOS, Chrome, Windows 11, or Arc browser frame. Set a custom title and URL, then export.

### 📐 Canvas Presets
One-click resize to Twitter, Instagram, YouTube, LinkedIn, and Story dimensions. Annotate once, export at the right size.
